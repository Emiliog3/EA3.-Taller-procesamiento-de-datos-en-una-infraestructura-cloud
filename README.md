# EA3 - Taller: procesamiento de datos en una infraestructura cloud

Este repositorio contiene la entrega de la Unidad 3, Evidencia de Aprendizaje EA3, correspondiente al taller de procesamiento de datos en una infraestructura cloud usando Databricks Community Edition, Apache Spark, Spark SQL y tabla Delta.

## Archivo principal

- Gomez_Pablo_Actividad_2.ipynb

Aunque la plataforma conserva la nomenclatura Actividad_2 para el archivo, el desarrollo corresponde conceptualmente a la Actividad 03 / EA3 de Big Data.

## Contenido del notebook

El notebook incluye:

1. Diseño del esquema de datos.
2. Configuración y evidencia del entorno en Databricks CE.
3. Ingesta del dataset Titanic y creación de tabla Delta.
4. Validaciones con PySpark y Spark SQL.
5. Comparación entre SQL y Spark.
6. Matriz final de cumplimiento frente a la rúbrica.
